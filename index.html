How it Works:

Key Principle: Higher GPi activity → More GABA release → More thalamic inhibition → Less movement
Direct Pathway: Cortex → Striatum (GABA) inhibits GPi → Lower GPi activity → Less thalamic inhibition → More movement
Indirect Pathway: GPe (GABA) inhibits STN → STN (glutamate) excites GPi → Higher GPi activity → More thalamic inhibition → Less movement
Balance: Normal movement requires balanced GPi activity from competing direct/indirect influences
